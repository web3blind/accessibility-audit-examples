# Weekly accessibility quests

Формат для регулярных «полуквестов» по доступности.

## Rhythm

1. **Понедельник–суббота:** собираем кандидатов.
2. **Воскресенье:** выбираем кейс(ы), делаем полный агентный аудит, готовим короткую сводку для канала и сохраняем полный отчёт в этом репозитории.
3. **После публикации:** если возможно, открываем upstream PR/issue или готовим contact draft.
