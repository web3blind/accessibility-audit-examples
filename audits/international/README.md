# International accessibility audit examples

Примеры по проектам вне российской/русскоязычной подборки.

Здесь можно хранить такие же кейсы, как в `audits/russian/`:

```text
audits/international/<case-slug>/
├── audit.md
├── summary.md
├── findings.json
├── evidence.md              # для сайтов или больших проектов, если нужно
└── issue-or-pr.md           # ссылка/черновик upstream follow-up
```

## Candidate criteria

- Реальный open-source проект или публичный сайт.
- Не sensitive-first тема: избегать оружия, экстремизма, adult, gambling, scam/murky finance и сомнительной медицины.
- Достаточно UI-поверхности для полезного аудита: формы, модалки, списки, медиа, мобильные/desktop-компоненты.
- Желательно, чтобы можно было открыть небольшой PR или issue.
