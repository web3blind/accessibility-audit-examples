# Codeversehub Website — skip link

**Type:** Website / React  
**Target:** https://github.com/TheCodeVerseHub/Codeversehub-Website  
**Upstream:** https://github.com/TheCodeVerseHub/Codeversehub-Website/pull/15

## Main issue

Keyboard and screen-reader users lacked a direct way to bypass repeated navigation and jump to the main content.

## Fix/proposal

Add a focus-visible skip link and main-content targets on key pages.

## Why it matters

This is a concrete accessibility example from an upstream contribution: the visible interface behavior needed a programmatic equivalent for keyboard and/or screen-reader users.
